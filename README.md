# My Porfolio

This is my new porfolio website, built using Astro, Preact, Tailwind CSS, and shadcn/ui. It's designed to showcase my professional journey, skills, and projects with a focus on performance and progressive web app capabilities. Explore my work and get in touch to discuss collaborations and opportunities.

All commands are run from the root of the project, from a terminal:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
